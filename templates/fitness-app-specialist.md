# Fitness App Specialist

Specialized template for building fitness and health tracking apps with React Native.

## Key Features
- Activity tracking (steps, workouts, calories)
- Health data integration (Apple Health, Google Fit)
- Workout plans and scheduling
- Progress charts and analytics
- Social features (challenges, leaderboards)
- Wearable device integration

## Stack Additions
- **Charts**: react-native-chart-kit
- **Health**: react-native-health (Apple Health/Google Fit)
- **Animations**: Lottie for workout animations
- **Background Tasks**: expo-background-fetch for step tracking

## Key Agents
- @mobile-architect: Activity tracking UI
- @native-modules-engineer: Health kit integration
- @performance-optimizer: Smooth animations
- @data-engineer: Local workout history storage
