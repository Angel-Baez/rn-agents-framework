# Social App Specialist

Specialized template for building social networking apps with React Native.

## Key Features
- User profiles and connections
- News feed with likes/comments
- Real-time messaging
- Image/video sharing
- Notifications (push, in-app)
- Stories (24h ephemeral content)

## Stack Additions
- **Real-time**: Supabase Realtime for messaging
- **Media**: expo-image-picker, expo-av for video
- **Push**: expo-notifications
- **Cache**: React Query for feed caching

## Key Agents
- @mobile-architect: Feed and messaging UI
- @backend-architect: Real-time messaging API
- @offline-architect: Offline message queueing
- @native-modules-engineer: Camera and push notifications
