# Fintech App Specialist

Specialized template for building fintech and banking apps with React Native.

## Key Features
- Account management
- Transaction history
- Money transfers
- Bill payments
- Budgeting and analytics
- Biometric authentication

## Stack Additions
- **Biometrics**: expo-local-authentication
- **Security**: Enhanced SecureStore usage
- **Charts**: react-native-chart-kit for financial data
- **Encryption**: Additional encryption for sensitive data

## Key Agents
- @security-guardian: CRITICAL - financial data security
- @mobile-architect: Financial dashboard UI
- @backend-architect: Secure transaction APIs
- @test-engineer: Comprehensive testing for transactions

## Security Requirements
- Two-factor authentication (2FA)
- Biometric login
- Session timeout
- Transaction limits
- Fraud detection
