# E-commerce Mobile Specialist

Specialized template for building e-commerce shopping apps with React Native.

## Key Features
- Product catalog with search/filters
- Shopping cart and checkout
- Payment integration (Stripe, PayPal)
- Order tracking
- Push notifications for order updates
- Wishlist and favorites

## Stack Additions
- **Payments**: @stripe/stripe-react-native
- **Search**: Algolia or Supabase full-text search
- **Analytics**: Amplitude for conversion tracking
- **Deep Linking**: For product sharing

## Key Agents
- @mobile-architect: Product catalog and cart UI
- @backend-architect: Payment processing API
- @security-guardian: Payment data security
- @data-engineer: Order and cart persistence
